# minecraft-steve-voice-pack
マイクラのスティーブの声を復活させることができるリソースパックです。

バージョンを変えたい人は[pack.mcmeta](./pack.mcmeta)ファイルの`"pack_format"`の値を変更してください。

```JSON
{
    "pack": {
        "pack_format": 13,
        "description": "プレイヤーのダメージ音をスティーブのボイスに置き換えるリソースパックです。"
    }
}
```
変更する数値はWikiなどで各自調べてください。
